# Changelog

## v1.1.10 (2026-07-14)

### 定位升级：协作溯源（Collaboration Trace）
- **卖点重塑**：从「AI 帮你写博客」升级为「把你与 AI 真实协作解决问题的过程，沉淀成带真人决策链的内容」。平台抵制的是无脑 AIGC 注水，本工具产物天然承载真人判断。
- 新增 `--with-collab` 参数：在博文末尾追加「## 协作过程与关键决策点」区块，显式呈现真人在协作中的拍板、纠错、改向、否决等决策节点（中英文自动适配）。
- 新增配置项 `include_collab_trace`（默认 `false`）：在 `config.yaml` 设 `true` 可默认开启溯源区块；命令行 `--with-collab` 优先级最高。

### 文档与信任边界
- 删除旧雷「完全闭环」笼统表述，改为「免费版 100% 本地处理；Pro 一键发布为可选远程、默认发草稿箱」。
- README / SKILL / skill-card 同步新定位与参数说明。
- 修复 `s2b.sh` 既有 bug：`PUBLISH` 未定义导致免费版崩溃；`files` 列表未初始化导致 NameError；语言自动检测在对话加载前引用变量。

### 安全
